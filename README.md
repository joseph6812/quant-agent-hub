# QuantAgent Hub - AI智能体量化策略分享社区

一个专为AI智能体量化策略开发者打造的分享和交流平台。

## 项目概述

QuantAgent Hub 是一个开源的量化策略分享社区，专注于：
- AI智能体量化策略的分享与下载
- 回测结果的展示与交流
- 策略开发者之间的经验分享

## 核心功能

### 已完成功能
- ✅ 首页展示 - 精选策略、统计数据、风险提示
- ✅ 策略广场 - 策略列表、搜索筛选
- ✅ 策略详情 - 回测指标、代码展示、评论区
- ✅ 策略上传 - 完整的策略提交表单
- ✅ 免责声明 - 完整的法律风险提示

### 页面结构
```
/                    - 首页
/strategies          - 策略广场
/strategies/[id]     - 策略详情页
/upload              - 上传策略
/disclaimer          - 免责声明
```

## 技术栈

- **框架**: Next.js 16 + React 19
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **组件**: shadcn/ui
- **构建**: 静态导出（Static Export）

## 本地开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 部署

项目配置为静态导出，可部署到任何静态托管平台：
- Vercel（推荐）
- GitHub Pages
- Netlify
- 阿里云OSS
